The software I use

This is the mostly complete list of the software I use semi-daily. This list in alphabetical order. Do yourself a favor and install 1Password first. Last updated: Feb 2020

Design

  • Adobe Illustrator – OG design tool
  • Adobe Photoshop – OG design tool's brother
  • Figma – In browser design tools
  • Sketch – Yet another design tool

Development

  • Homebrew – Package management
  • Node – Because Javascript
  • NPM – Less preferred javascript package managment (freaking javascript)
  • Postegres.app – Postgres databases made easy
  • Postico – Postgres database client
  • Postman – Http Client for the API's
  • Rbenv – Ruby gem management
  • SequalPro – MySQL database client

  • Transmit – Old school coding needs
  • VScode – Code IDE
  • Xcode – Nobody like this but we all have to use it

Leisure

Productivity

  • Alfred – Spotlight replacement with automations and more.
  • Calcbot – The best digital calculator out there
  • Cloudapp – A quick screenshot and file sharing utility
  • Divvy – Keyboard accessible window management
  • Fantastical – The calendar Apple should have made
  • Pastebot – Clipboard management
  • Notion – MArkdown notes and document sharing
  • Slack – quick communication tool
  • Spark – Modern email client

Security

Browser extensions
